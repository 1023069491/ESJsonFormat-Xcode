# ESJsonFormat-Xcode
将Json格式化输出为模型的属性

怎么安装：

下载-Xcode打开-command+B-重启Xcode

怎么使用：

window菜单-ESJsonFormat-输入json-enter

或者：

control+shift+j-输入json-enter


功能说明：

-0.1

1.通过Json字符串生成对应属性

2.通过文件写入的方式生成到.m文件

3.支持输入嵌套模型名称

注：目前只支持Objective-C
